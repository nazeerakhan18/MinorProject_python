# 📱 GroupDNA - WhatsApp Chat Analyzer

## 📌 Overview

GroupDNA - WhatsApp Chat Analyzer is a Python-based data analytics project that analyzes exported WhatsApp chat data and provides meaningful insights about group activity, participant behavior, and messaging patterns.

This project was developed as a Minor Project for the Integrated M.Sc. Computational Statistics and Data Analytics program.

---

## 🚀 Features

* Read and process exported WhatsApp chat files
* Parse chat messages into structured data
* Count total messages and participants
* Identify the most active participant
* Find the most active day and hour
* Perform word frequency analysis
* Generate a 24×7 activity matrix using NumPy
* Detect personality traits such as:

  * 🏆 Spammer
  * 👻 Ghost
  * 🌙 Night Owl
* Generate a final summary report
* Save the analysis report as a text file
